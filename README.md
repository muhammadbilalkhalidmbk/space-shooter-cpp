# 🚀 Space Shooter Game

A 2D space shooter game built in **C++ using the SFML library**, developed as a 2nd semester OOP project at Air University.

---

## 🎮 Gameplay Features

- 🛸 Player-controlled spaceship with shooting mechanics
- 👾 Enemy waves with increasing difficulty
- 💥 Boss battle with unique bullet patterns
- ⚡ Power-ups: Multi-gun, Health restore, Drone companion
- 🔊 Sound effects for shooting and explosions
- 🎨 Custom sprites and background

---

## 🏗️ OOP Concepts Used

- **Classes & Objects** — Spaceship, Enemy, Boss, PowerUp, Bullet
- **Inheritance** — HangmanPlayer extends Player base class
- **Encapsulation** — Each game entity manages its own state
- **Polymorphism** — Different bullet and enemy behaviors

---

## 🛠️ Tech Stack

- **Language:** C++
- **Library:** SFML 2.x (Simple and Fast Multimedia Library)
- **IDE:** Visual Studio

## ⚙️ How to Run

1. Install [SFML 2.x](https://www.sfml-dev.org/download.php) on your system
2. Clone this repository:
```bash
   git clone https://github.com/muhammadbilalkhalidmbk/space-shooter-cpp.git
```
3. Open in Visual Studio and link SFML libraries
4. Build and run the project
